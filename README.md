# Weekend Getaway Ranker

## Overview
This project ranks weekend travel destinations based on city-wise attractions using ratings, popularity, and visit duration.

## Dataset Details
The dataset contains Indian tourist places with attributes such as city, rating, popularity, visit duration, and significance.

## Ranking Criteria
Since distance data is unavailable, destinations are ranked using:
- Google Review Rating (40%)
- Popularity (Number of Reviews) (40%)
- Time Needed to Visit (20%)
